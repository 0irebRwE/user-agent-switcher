User Agent Switcher

The User Agent Switcher extension adds a menu and a toolbar button to switch the user agent of a browser. The extension is available for Firefox, and will run on any platform that this browser supports including Windows, OS X and Linux.

The extension can be installed from its home page:

  • Home page: http://chrispederick.com/work/user-agent-switcher/

Or directly from the Firefox extension repository:

  • Firefox: https://addons.mozilla.org/en-US/firefox/addon/user-agent-switcher/
